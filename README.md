A simple interactive quiz app built with **HTML, CSS, and JavaScript**.  

Test your knowledge with multiple-choice questions and see your score at the end!

## Features

- Display questions one by one  
- Multiple-choice answers  
- Highlights correct and incorrect answers  
- Progress bar to show quiz completion  
- Shows final score at the end  

## How to Use

1. Open `index.html` in your browser  
2. Read the questions and click the answer you think is correct  
3. Click **Next** to go to the next question  
4. View your final score at the end  

## Tech Stack

- HTML  
- CSS  
- JavaScript
